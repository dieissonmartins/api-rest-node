##  👨‍💻  Como baixar o projeto

```bash
    
    # Clonar repositório
    $ git clone https://github.com/dieissonmartins/api-rest-node.git

    # Entrar no diretório
    $ cd backend

    # Instalar as dependências
    $ yarn install

    # Rodar servidor 
    $ yarn dev  
```
